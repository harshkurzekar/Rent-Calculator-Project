# Rent Calculator - Automated Expense Distribution System

## Project Overview
The Rent Calculator is a Python console application designed to simplify the fair distribution of shared monthly expenses among roommates. It helps calculate individual shares for rent, electricity, and other utilities, reducing disputes and saving time.

## Features
- Calculates total electricity bill based on units consumed and rate per unit.
- Evenly splits rent and other shared expenses among roommates.
- Supports multiple calculations in one session.
- Simple text-based interface for easy use.
- Helps avoid manual errors in expense splitting.

## Motivation
Living in shared accommodations often leads to arguments over expense sharing. This project provides an automated way to share costs fairly, making co-living smoother and more transparent.

## How to Use
1. Run the Python script `rent_calculator.py`.
2. Enter the number of roommates.
3. Enter the rent amount, food expense, electricity units, and rate per unit as prompted.
4. View the calculated share each roommate must pay.
5. Choose to calculate again or exit.
